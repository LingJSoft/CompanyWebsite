SS

Soil Science Dictionary

Soil science dictionary is a dictionary app that contains soil science related terminologies. It's a completely open source app which is developed for the educational and informational purpose only. Our main goal is to make a collection of the soil terminologies so that the user can easily find it.

Features:

  • Self-expandable database
  • Full customizable database
  • Random Word
  • Flashcards
  • Add Custom Word Option
  • Night Mode
  • Backup Data
  • Voice Search
  • Text To Speach

This app uses the self-expandable database, this will expand with user engagement. And also this database is fully customizable. As a user, you can customize the word details with your own. And also have the option to add your own word. By default, this word will be shared with other users, but you can turn this off from settings.

Another great feature is the random word, the user can easily get a random word every time you click, so never stop learning. Flashcards for memorizing the word. And also favorite word list.

The user can search with their voice and text to speech is also available. The user can able to use this app with night mode and also can able to change the theme.

And some more features are waiting. Install and start using the app.

WT

Weak Timer

Week timer is for tracking week working hour with completion rate. Track time for individual work and also total work in a week. See the progress of your total week work budget. Also can compare with previous records.

In short: Plan your week work time budget and track the complementation rate. You can track your working time details for both individual task and total completion time in a week. More features are also available. Use the app and track the time for individual work and boost your productivity.

Features:

  1. 1.Records previous week task time, for the individual. you can find the details in which week you work best in which task.
  2. 2.you can track your productivity rate in a graph for the total working time in the week.
  3. 3.you can also able to track your productivity rate in a graph for the individual working time in the week.
  4. 4.you can able to find the percentage of per task in a week (on the budget)
  5. 5.You can also archive work. (that value is also saved)
  6. 6.Bunch of customization available in settings. For instance
  7. 7.Pause tracking
  8. 8.start the week on a selected date and so on.
  9. 9.Change the theme with your favorite color.
  10. 10.Material design and attractive UI.
  11. 11.Optimized design for both tablet and phone.

How to use the app:

Make a budget of your total working time for individual work in a week. And input all task with time budget in the app. At the end of the week,

You will find:

  1. 1.week total complete time with the budget
  2. 2.graph of total weeks complete rate
  3. 3.graph of individual task
  4. 4.also previous week complete details.

So why you are waiting for? Download and install the app. Use your time properly and boost productivity.
